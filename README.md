# ReKurn Versioning System

Rekurn is a custom versioning system I built from scratch as a Git alternative. I created it to have full control over our codebases and applications on oreulius.com. It serves as our own independent repository pull and push system, and also a general open source versioning system with no reliance on GitHub or other external platforms. Rekurn works like Git in spirit with familiar commands like commit, branch, and push. However it is designed specifically for modern JavaScript and TypeScript projects and Vercel hosted sites. It is lightweight, open source, and built to run securely inside our own infrastructure.
